1. src/content.config.ts 생성 — CONTEXT.md의 locked decisions을 반영한 Zod 스키마:
   ```
   - title: z.string()
   - description: z.string()
   - category: z.enum(['finance', 'marketing', 'hr', 'it'])  // 업종 4개
   - useCase: z.string()  // 용도: 보고서, 수집, 알림 등 (자유 텍스트)
   - difficulty: z.enum(['beginner', 'intermediate', 'advanced'])  // 난이도 3단계
   - pricing: z.enum(['free', 'premium'])  // 가격 모델
   - estimatedMinutes: z.number().positive()  // 분 단위 설정 시간
   - credentials: z.array(z.string())  // 필요 크레덴셜 배열
   - tags: z.array(z.string()).default([])  // 한국서비스 등 태그
   - workflowJsonPath: z.string()  // public/workflows/ 내 JSON 경로
   - nodeCount: z.number().optional()
   - publishedAt: z.coerce.date()
   - updatedAt: z.coerce.date().optional()
   ```
   glob 로더 사용: `glob({ pattern: "**/*.mdx", base: "./src/content/templates" })`
   Astro 내장 zod 사용: `import { z } from 'astro/zod'`
