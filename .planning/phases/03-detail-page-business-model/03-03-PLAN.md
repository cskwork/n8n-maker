   - 빌드 타임에 `fs.readFileSync(path.join(process.cwd(), 'public', workflowJsonPath), 'utf-8')`로 워크플로우 JSON 읽기.
   - nodes 배열에서 `{ name, type }` 추출. type에서 `n8n-nodes-base.` prefix 제거, `@n8n/n8n-nodes-langchain.` prefix를 `AI:` 로 변환.
   - connections 객체에서 연결 관계 파싱하여 노드 간 순서를 결정 (source -> target 순서).
   - 노드를 좌->우 플로우차트로 표시: 각 노드를 카드(`px-3 py-2 bg-white border border-surface-200 rounded-lg shadow-sm`)로, 사이에 화살표 SVG로 연결.
   - 모바일에서 flex-wrap으로 줄바꿈 허용.
   - try-catch로 JSON 파싱 실패 시 빈 노드 배열 처리.
   - 상단에 "워크플로우 미리보기 (간소화)" 안내 텍스트.

2. **src/components/detail/WorkflowDiagram.astro** — n8n-demo CDN 임베딩 컨테이너.
   - Props: `{ workflowJsonPath: string; templateTitle: string }`
   - 3단계 표시: 스켈레톤(기본) -> n8n-demo(로드 성공) -> 폴백(로드 실패).
   - 컨테이너: `rounded-xl border border-surface-200 overflow-hidden bg-white shadow-sm`.
   - **스켈레톤** (`id="diagram-skeleton"`): `h-[300px] md:h-[450px] bg-surface-50 animate-pulse flex items-center justify-center`. "워크플로우 다이어그램 로딩 중..." 텍스트.
   - **n8n-demo 영역** (`id="diagram-live"`, 초기 hidden): `h-[300px] md:h-[450px]`. 내부에 n8n-demo 요소 — collapseformobile="false" clicktointeract="false" hidecanvaserrors="true" tidyup="true".
   - **폴백 영역** (`id="diagram-fallback"`, 초기 hidden): slot name="fallback"으로 DiagramFallback을 부모에서 주입.
   - **CDN 스크립트 로딩**: 3개 script is:inline src 태그 (webcomponents-loader.js, lit polyfill, n8n-demo.bundled.js). 반드시 is:inline 사용하여 Astro 번들링 방지.
   - **클라이언트 스크립트** (script define:vars):
     - Props에서 `const fullPath = import.meta.env.BASE_URL + workflowJsonPath;`를 계산하고 이를 define:vars로 전달.
     - `fetch(fullPath)` -> JSON 파싱 -> `demo.setAttribute('workflow', JSON.stringify(workflow))` -> 스켈레톤 hidden, live 표시.
     - 5초 타임아웃 설정: n8n-demo의 iframe 로드를 기다리되, 5초 후 응답 없으면 폴백으로 전환.
     - catch 시: 스켈레톤 hidden, 폴백 표시.
   - **모바일 터치 지원 (Locked Decision)**: 컨테이너에 overflow-auto CSS. style="touch-action: pan-x pan-y pinch-zoom;" 인라인 스타일로 핀치줌 허용.
