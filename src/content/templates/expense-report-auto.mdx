## 이 워크플로우는

Webhook을 통해 경비 내역(날짜, 항목, 금액, 사용자, 비고)을 접수받아 Google Sheets에 자동으로 기록합니다.
접수가 완료되면 Slack 채널로 확인 알림을 보내고, Webhook 응답으로 처리 결과를 반환합니다.

팀 경비 관리를 수기 엑셀 입력 대신 자동화하여, 실시간 경비 현황 파악과 빠른 접수 확인이 가능합니다.
모바일에서도 Webhook URL로 간편하게 경비를 제출할 수 있습니다.

## 설정 가이드

### Step 1: 크레덴셜 발급

Google Sheets OAuth: n8n에서 Google OAuth2 크레덴셜을 설정합니다. Sheets 읽기/쓰기 권한이 필요합니다. Gmail OAuth: 보고서 이메일 발송을 위해 Gmail 크레덴셜을 설정합니다. Slack OAuth: 워크스페이스에서 Bot Token을 발급받습니다.

### Step 2: n8n 설정

다운로드한 JSON 파일을 n8n의 Workflows 메뉴에서 Import합니다. Google Sheets에 경비 기록용 시트를 만들고 컬럼(날짜, 항목, 금액, 사용자, 비고)을 설정합니다. 각 노드에 크레덴셜을 연결합니다. Webhook URL은 워크플로우 활성화 후 자동 생성됩니다.

### Step 3: 활성화

워크플로우를 활성화하면 Webhook URL이 생성됩니다. 이 URL로 POST 요청을 보내면 경비가 자동으로 접수됩니다. 요청 본문에 date, item, amount, user, note 필드를 JSON으로 전송하면 됩니다.
