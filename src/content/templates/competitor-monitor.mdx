## 이 워크플로우는

네이버 뉴스 API로 경쟁사 관련 최신 뉴스를 자동 수집하고, OpenAI가 핵심 동향을 3줄로 요약하여 Slack 채널로 브리핑합니다.

마케팅팀의 경쟁사 동향 파악을 자동화하여, 매일 뉴스를 검색하는 시간을 절약할 수 있습니다. 평일 오전 9시와 오후 2시에 자동 실행됩니다.

## 설정 가이드

### Step 1: 크레덴셜 발급

[네이버 개발자센터](https://developers.naver.com)에서 애플리케이션을 등록하고 Client ID와 Client Secret을 발급받습니다. 검색 API 사용 권한을 선택합니다. OpenAI API 키를 발급받고, Slack OAuth 앱을 설정합니다.

### Step 2: n8n 설정

다운로드한 JSON 파일을 n8n에서 Import합니다. n8n 환경변수에 `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`을 설정합니다. "네이버 뉴스 검색" 노드에서 검색 키워드를 모니터링할 경쟁사명으로 변경합니다. OpenAI와 Slack 크레덴셜을 각 노드에 연결합니다.

### Step 3: 테스트 및 활성화

수동 실행하여 뉴스 검색 -> AI 분석 -> Slack 전송이 정상 동작하는지 확인합니다. 활성화하면 평일 오전/오후에 자동으로 경쟁사 동향 브리핑을 받을 수 있습니다.
