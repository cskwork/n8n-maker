## 이 워크플로우는

Webhook으로 신규 입사자 정보를 접수하면 환영 이메일 발송, Slack 팀 채널 안내, Google Sheets 온보딩 체크리스트 생성을 한 번에 자동으로 처리합니다.

HR 담당자가 반복하는 온보딩 업무를 자동화하여, 입사자 경험을 개선하고 업무 누락을 방지합니다.

## 주요 기능

- **자동 환영 이메일**: 입사자 이름, 부서, 입사일이 포함된 맞춤형 환영 메일 발송
- **Slack 팀 공지**: 팀 채널에 신규 입사자 소개 메시지 자동 전송
- **체크리스트 생성**: Google Sheets에 계정 생성, 장비 지급, 교육 일정 등 온보딩 항목 자동 생성
- **HR 완료 알림**: 온보딩 프로세스 시작 알림으로 후속 조치 안내

## 설정 가이드

1. JSON 파일을 n8n에 import합니다
2. Gmail OAuth, Slack OAuth, Google Sheets OAuth 인증을 각각 설정합니다
3. Slack 채널 ID를 팀 채널과 HR 채널로 설정합니다
4. Google Sheets에 온보딩 체크리스트 스프레드시트를 생성합니다
   - 컬럼: 이름 / 부서 / 입사일 / 계정생성 / 장비지급 / 교육일정
5. Webhook URL로 테스트 요청을 보내 동작을 확인합니다:
   ```json
   { "name": "홍길동", "email": "hong@company.com", "department": "개발팀", "startDate": "2026-04-01" }
   ```
