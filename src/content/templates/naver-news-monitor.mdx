## 이 워크플로우는

네이버 뉴스 검색 API로 원하는 키워드의 최신 뉴스를 업무시간(오전 9시, 오후 1시, 오후 6시)마다 자동 수집하여 Slack으로 전송합니다.

기술 트렌드, 경쟁사 동향, 업계 뉴스를 실시간으로 모니터링하고 싶은 마케터, 기획자, 개발자에게 유용합니다.

## 주요 기능

- **키워드 기반 뉴스 검색**: 네이버 뉴스 API로 최신순 검색
- **HTML 태그 자동 제거**: 깔끔한 텍스트로 정리
- **조건부 알림**: 새 뉴스가 있을 때만 Slack 전송
- **평일 업무시간 자동 실행**: 하루 3회 자동 체크

## 설정 가이드

### Step 1: 네이버 API 키 발급

1. [네이버 개발자센터](https://developers.naver.com)에 가입합니다
2. **애플리케이션 등록**을 클릭합니다
3. 애플리케이션 이름을 입력하고 **검색** API 사용을 선택합니다
4. 등록 후 **Client ID**와 **Client Secret**을 확인합니다
5. API 키는 즉시 발급되며, 별도 승인 절차가 없습니다

### Step 2: n8n 설정

1. JSON 파일을 n8n에 import합니다
2. n8n 환경변수에 다음을 설정합니다:
   - `NAVER_CLIENT_ID`: 네이버 Client ID
   - `NAVER_CLIENT_SECRET`: 네이버 Client Secret
3. Slack OAuth 인증을 설정합니다
4. "네이버 뉴스 검색" 노드에서 검색 키워드를 원하는 것으로 수정합니다

### Step 3: 테스트 및 활성화

1. 수동 실행으로 검색 결과가 정상적으로 나오는지 확인합니다
2. Slack 채널에 뉴스 알림이 도착하는지 확인합니다
3. 스케줄을 활성화합니다

### 참고사항

- 일일 **25,000회** 호출 제한 (하루 3회 실행이면 충분합니다)
- 한 번에 최대 **100개** 결과를 가져올 수 있습니다
- 검색어에 따옴표를 사용하면 정확한 문구 검색이 가능합니다
